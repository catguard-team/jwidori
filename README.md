# jwidori

> 첫 쥐돌이 — 새끼고양이가 잡은 첫 작품 갤러리입니다.

안녕하세요. 처음으로 만든 코드 한 줄, 처음 켠 웹페이지, 처음 그린 픽셀, 처음 셋업한 라즈베리파이. 모두 작아도 괜찮습니다. **첫 쥐돌이는 부끄러워하지 않습니다** (행동강(綱)냥(領) 8조).

## 올리는 법

### 0. 처음이시면

GitHub이 처음이세요? [`saenoyangi-handbook/GITHUB-GUIDE.md`](https://github.com/catguard-team/saenoyangi-handbook/blob/main/GITHUB-GUIDE.md)를 먼저 봐주세요.

### 1. 보호자 동의 (필수)

> [!IMPORTANT]
> 미성년자분이거나 결정 보조가 필요한 학습자분은 **공개 전 보호자 동의**가 별도로 필요합니다.
> 작성 동의와 공개 동의는 다릅니다. [`SAFETY-NOTES §3-2`](https://github.com/catguard-team/saenoyangi-handbook/blob/main/SAFETY-NOTES.md#3-2-결과물-공개-전-핵심)를 확인해 주세요.

### 2. issue 또는 PR

| 형태 | 누가 |
|------|------|
| **Issue로 신청** ([New Issue](https://github.com/catguard-team/jwidori/issues/new)) | 본인이 PR 올리시기 어려우실 때. 멘토 자경단원이 대신 올려드립니다 |
| **PR로 직접 등재** | GitHub이 익숙하신 자경단원·보호자분 |

내용:
- 첫 쥐돌이 한 줄 설명
- 어느 트랙·어느 챕터에서 잡으셨는지
- 결과물 링크 또는 스크린샷 (식별 정보는 빼주세요)
- 보호자 동의를 받으셨는지 확인 ([SAFETY-NOTES](https://github.com/catguard-team/saenoyangi-handbook/blob/main/SAFETY-NOTES.md))
