# jwidori

> 첫 쥐돌이 — 새끼고양이가 잡은 첫 작품 갤러리

처음으로 만든 코드 한 줄, 처음 켠 웹페이지, 처음 그린 픽셀, 처음 셋업한 라즈베리파이. 작아도 됩니다. **첫 쥐돌이는 부끄러워하지 않습니다** (행동강(綱)냥(領) 8조).

---

## 무엇을 올리나

| 카테고리 | 예시 |
|---------|------|
| 🎮 첫 게임 | Scratch / 엔트리 / 코드.org 미니 게임 |
| 🌐 첫 웹 | HTML 자기소개, 메모 페이지 |
| 🐍 첫 코드 | Python 5줄, JS 한 함수 |
| 🍎 첫 셋업 | macOS·Windows·라즈베리파이 본인 환경 사진 |
| 📝 첫 문서 | 자경단 PR (오타 한 글자라도) |
| 🛒 첫 가게 | 자영업자 트랙 — 매장 안내 QR, 휴무 공지 자동화 |

> 분류 안 맞으면 그냥 올리세요. 칸막이 없습니다.
